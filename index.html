Data Sources

資料源不是只有一個，要分層查才準。

公共工程、建管、道路挖掘、環評、都市計畫各有自己的資料入口。本網站先把入口整理成可維護架構，避免資料亂成一鍋粥。

Maintain Like a Pro

維護方式：只改一個資料檔，地圖自動更新。

1. 新增工程

打開 data/projects.js，複製一筆工程資料，改名稱、類型、座標、金額、甲方、廠商、工期即可。No drama，維護超直覺。

2. 畫位置範圍

線型工程用 line，基地型工程用 polygon，單點工程用 point。座標格式是 [緯度, 經度]。

3. 資料品質

每筆工程都有 confidence 欄位：官方確認、部分確認、待補資料。讓使用者知道哪些資料很穩，哪些還要回查官方。

4. Vercel 部署

本網站是靜態網站，不需要 build 到 dist。Vercel 直接部署根目錄，速度快、故障點少、手機載入更輕。
